📚 Курс туралы
Бұл репозиторий ITECHRESEARCH мектебінің Машиналық оқыту курсының толық материалдарын қамтиды. Курс 26 апта бойы жалғасады және теориялық білім мен практикалық дағдыларды біріктіреді.
🎯 Курстың мақсаттары
<li> Машиналық оқытудың негізгі концепцияларын меңгеру <li>
<li> Классикалық және заманауи ML алгоритмдерін үйрену <li>
<li> Практикалық жобаларда тәжірибе жинау <li>
<li> Нақты есептерді шешу дағдыларын дамыту <li>

📊 Курс құрылымы

<li> Дәрістер: 26 апта (аптасына 1 дәріс)<li>
<li> Практикалық сабақтар: 52 сабақ (аптасына 2 практика) <li>
<li> Жобалар: 4 негізгі жоба <li>
<li>Тапсырмалар: Әр аптада үй тапсырмасы<li>
